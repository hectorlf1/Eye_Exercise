Title of the project: Eye Exercise

Description: This design consist of two eyes centered on the page. The eyeballs inside each eye will follow the mouse arrow anywhere it moves within the page.
 
How to run: Load the HTML file and move the cursor inside the page.

Roadmap of future improvements: some future enhancements include,
A) Have each eyeball move independently from one another so that they each both in a different direction if the cursor is in the middle of both.
B) Add an initiation button to the page.
